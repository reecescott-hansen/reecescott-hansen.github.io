April 2026 — Latest Update

Over the past few weeks, I have continued building my finance website and expanding it into a much larger project. What started as a simple homepage has now grown into a six-page website covering stock analysis, my mock portfolio, AI and markets, project updates, and a new freelance web design page.

One of my main goals this month was to make the website look more professional. I updated the design with a cleaner CSS stylesheet, improved fonts, better card layouts, and a consistent color scheme across every page. I also fixed several bugs in the original code including a broken CSS bracket and spelling errors in the About section.

I also launched a new Hire Me page where I am offering freelance web design services to local businesses and people who need a website. This is something I want to grow into a real side business — designing and building websites for clients while I am still in school.

Going forward, my goals are to land my first freelance client, keep updating my stock analysis monthly, and continue learning how investing, markets, and web design all connect to my bigger goals in finance.

March 2026 — Getting Started

This was the month I launched the first version of this website. I built it from scratch using HTML and CSS skills I learned in Web Design class at Columbia Grammar. My initial goal was simple — create a space where I could track the stocks I was researching and document what I was learning about investing.

In March I added the first six stock analysis cards, built the basic structure of the site including the header, navigation bar, footer, and market snapshot ticker. It was my first time building a complete multi-page website on my own.
